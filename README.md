# Groupe3_EtudeGrapheDeMarkov

## Configuration du projet dans CLion

**IMPORTANT :** Après avoir ouvert le projet, suivez ces étapes :

1. Allez dans **Run → Edit Configurations...**
2. Dans le champ **Working directory**, mettez : `$ProjectFileDir$`
3. Cliquez sur **OK**
4. Lancez le projet

Sans cette étape, les fichiers ne seront pas trouvés sur l'ordinateur
